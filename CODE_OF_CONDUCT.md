# Code of Conduct

Cook Editor adopts the [Contributor Covenant, version 2.1][cc] as its Code of
Conduct. By participating in this project — through issues, pull requests,
discussions, or any other community channel — you agree to abide by it.

The full text is published at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

If you experience or witness behavior that violates this Code of Conduct,
please report it to **dubovskoy.a@gmail.com**.

Reports are handled in confidence. Project maintainers are obligated to
respect the privacy and security of the reporter of any incident.

## Enforcement

Project maintainers will follow the Contributor Covenant's
[Enforcement Guidelines][guidelines] when responding to reports. Maintainers
who do not follow or enforce this Code of Conduct in good faith may face
temporary or permanent repercussions as determined by other maintainers.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
[guidelines]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
